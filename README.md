# Python_Homework
## Урок 10. Создание отчетов в Allure Report
### Запуск тестов для формирования отчетов
Для запуска тестов выполните следующие действия:
1. Перейти в папке Урока 10. Введите в Терминале команду: cd Lesson_10;
2. Подключите Allure: pip install allure-pytest;
3. Запустите тесты и укажите путь к каталогу результатов тестирования: pytest -k test_10 --alluredir allure-result.

### Просмотр сформированного отчета
Для просмотра сформированного отчета выполните следующее:
1. Установите Allure Report с сайт (https://allurereport.org/docs/v3/install/);
2. В Терминале введите команду: allure serve allure-result.
